# Stock Tracking App Research

## Core Requirements
- Dark mode
- Bloomberg-meets-Apple aesthetic
- Tickers: AAPL, AMZN, NVDA, TSLA, BTC-USD
- Benchmark: QQQ
- Performance metrics: 1-month, 12-month
- Persistence: Journal notes via localStorage
- Manual refresh only

## Tech Stack
- Vite + React
- Vanilla CSS
- Yahoo Finance API (via proxy or public endpoint)
